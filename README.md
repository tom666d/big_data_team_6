## Project Overview
Poor data quality costs organizations an estimated $12.9 million 
per year. This system automatically detects schema drifts, null 
anomalies, format inconsistencies, and statistical outliers using 
Apache Spark, then leverages GPT-4o-mini to propose remediation 
strategies with confidence scores. A human analyst reviews and 
approves all suggestions through an interactive Streamlit dashboard.

## Dataset Sources
